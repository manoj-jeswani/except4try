/api/all_questions

Get all the questions available on site.

Parameters

None

Return Values

  • List of all questions, along with their attributes like votes, tags, upvoters_list, asked_at, etc.

Example Request


import requests

url="http://127.0.0.1:8000/api/all_questions"
r = requests.get(url)
print(r.json())

/api/answers/<qid>

Given the question id in form of <qid> :: Retrieves all the answers of the question.

Parameters

None

Return Values

  • List of all the answers corresponding to the given question, along with their attributes like votes, upvoters_list, answered_at, answered_by, etc.

Example Request


import requests

url="http://127.0.0.1:8000/api/answers/4"
r = requests.get(url)
print(r.json())

/api/ask_question/

Post request to ask a question programatically.

Parameters

  • topic: topic of question.
  • detail: description of question.
  • tags_list: list of strings indicating the categories under which the question falls.

Return Values

  • Posts the question and returns it's echo to indicate successful post.

Example Request


import requests

url="http://127.0.0.1:8000/api/ask_question/"
data={'topic':'Template filters in Django','detail':'library not found error being displayed','tags_list':['Python','Django']}
r = requests.post(url,auth=('username', 'password'),data=data)
print(r.json())

Output


{'topic': 'Template filters in Django', 'asked_at': '2018-02-07 22:28:39.246626+00:00', 'detail': 'library not found error being displayed', 'ans_count': '0', 'vote_count': '0', 'asked_by': 'jesin', 'downvoters_list': '[]', 'edited_at': '2018-02-07 22:28:40.149666+00:00', 'upvoters_list': '[]', 'tags': "['Python', 'Django']"}
